# Exchanger Monorepo

## Servers

- [Dev](https://exchanger-dev.herokuapp.com/)
- Production

## Technology Stack

- Typescript
- React
- Ant Design
- Formik
- Yup
- Redux
- Saga
- Next
- Storybook
- Firebase
- Heroku (only for MVP)
- (GraphQL)

## Start

```bash
$ yarn install
$ yarn dev
```

## Lint

```bash
$ yarn lint
```

## Production

```bash
$ yarn build
$ yarn start -p <$PORT>
```

## Storybook

```bash
$ yarn storybook
```

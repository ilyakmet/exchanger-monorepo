# Exchanger Monorepo

## Technology Stack

- Typescript
- React
- Redux
- Saga
- Next
- Storybook
- Firebase
- (GraphQL)

## Start

```bash
$ yarn install
$ yarn dev
```

## Lint

```bash
$ yarn lint
```

## Production

```bash
$ yarn build
$ yarn start
```

## Storybook

```bash
$ yarn storybook
```
